==================== README ====================
QUATTRO, A CSOPESY EMULATOR BY LIMTIN, MACAYAN, MADRIDIJO, AND ONG (S09)

1. From Visual Studio, select open a project or solution.
2. Navigate to the CSOPESY_Emulator/OSEmulator folder.
3. Select the OSEmulator.slnx file to open the project in Visual Studio.
4. The main function is located in CSOPESY_Emulator/OSEmulator/main.cpp. Run the program by pressing the green play button on the top of the screen.
5. A console will pop up with "=== QUATTRO OS Emulator Main Menu ===". You may now type "initialize" and proceed from there.
